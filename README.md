# 现金牛台账 · Cash Cow Ledger

一套「质量现金流 + 低换手」的**向前跟踪观察指数（paper index）**。不是回测，不是荐股，不预测——每个交易变更留痕，跑输也照登。

> **定位红线（不可动）**
> 1. 主产品是**向前跟踪的观察指数**，从开账日起按规则记账。回测仅作参考，且所有回测输出必须标注「存在前视 / 幸存者偏差风险」。
> 2. **不自动交易、不接券商 API、不荐股、不预测方向。**
> 3. 数据缺失时明确标记 `data_incomplete`，**禁止静默填充或跳过**。
> 4. 所有筛选参数在 `config.yaml`，代码里不硬编码阈值。

## 五层漏斗（参数见 config.yaml）

| 层 | 干什么 | 堵的坑 |
|---|---|---|
| **L1 池子** | 标普500快照 + 手工全球白名单 | 池子取材 |
| **L2 防雷** | 剔除：利润远超经营现金流、应收增速>收入×2、净股本不降反升(回购被稀释吞掉)、净负债/EBITDA过高 | 造假与财技 |
| **L3 质量** | 须全满足：FCF连续≥7年为正且CV低、毛利率≥30%且10年不下行、ROIC≥12%、资产增速≤收入增速 | 周期顶部假现金牛、帝国建设 |
| **L4 估值** | FCF收益率≥10年美债 或 PE≤30（取宽松者） | 好公司买太贵 |
| **L5 数量** | 综合得分取前 15-30 | 集中押注 / 伪装指数 |

## 数据源

- **美股财务（主）**：SEC EDGAR companyfacts（免费、实时、XBRL 全历史≈2009+）。撑得起「FCF连续7年」「毛利率10年」。
- **非美股（手工）**：`data/global_watchlist.csv`。爱马仕等不在 EDGAR，基本面人工核对年报填入，标 `source` 与 `as_of`。
- **价格 / 估值**：yfinance（仅对通过质量层的幸存者取价，避开限流）。

EDGAR 的两个真坑已在 `src/` 内处理：① 收入等概念跨会计准则换标签（ASC 606）→ 多候选标签按年补洞；② 同一序列量纲会变 + 拆股跳变（股本）→ `rescale_1000` + `split_adjust` 双归一。

## 用法

```bash
./.venv/bin/python -m src.run_screen
```

产出到 `output/`：
- `candidates_<date>.csv` — 当前候选名单（人工核对用）
- `funnel_<date>.csv` — 全量台账，含每只在哪层被淘汰、为什么

数据体检（改动 EDGAR/metrics 后先跑）：
```bash
./.venv/bin/python tests/probe_edgar.py     # 原始年度序列
./.venv/bin/python tests/probe_metrics.py   # 派生信号 + 归一验证
```

## 决策纪律

任何成分变更必须在 `data/decisions_log.csv` 留痕（日期、动作、当时排名、当时价格、理由），否则不执行。例外条款（规则外持仓）≤3 只、每只≤8%、每年强制重审。

## 分阶段

- **Phase 1（本仓当前）**：数据管道 + L1-L5 筛选器 → 候选名单人工核对。
- **Phase 2**：观察指数向前跟踪 + 三/四基准对比（SPY主裁判 / QQQ参照 / COWZ·QUAL同类体检）+ 决策日志。
- **Phase 3**：归因报告 + 预警（哪些持仓接近剔除线、哪些新股接近入选线）。
- **Phase 4（最后）**：历史回测，仅作参考，全程标注偏差风险。
